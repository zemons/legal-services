# Legal Services — AI-Powered Online Law Firm

## Vision

สำนักงานทนายความออนไลน์ที่ใช้ AI ช่วยอำนวยความสะดวกตลอด workflow ตั้งแต่รับปรึกษา ตอบคำถาม ร่างเอกสาร จนถึงส่งต่อทนายจริง ผ่าน LINE OA เป็นช่องทางหลัก

---

## ช่องทาง

| ช่องทาง | ใช้ทำอะไร |
|---------|----------|
| **LINE OA Bot** | ตอบคำถาม, คุยกับ AI, แจ้งเตือนนัด |
| **LIFF** | กรอก intake form, upload เอกสาร, ดู status |
| **Web Admin** | ทนาย/admin จัดการ case, upload เอกสารกฎหมาย |

---

## ความสามารถของระบบ

### AI ทำได้ทันที (ไม่ต้องมีทนาย)

- ตอบคำถามกฎหมายทั่วไปโดยอ้างอิงฎีกา / กฎหมายจริง
- อธิบายขั้นตอน / อายุความ / ค่าธรรมเนียม
- คัดกรองประเภทคดี และแนะนำทนายที่เหมาะสม
- Draft เอกสารเบื้องต้น (สัญญา, หนังสือมอบอำนาจ, หนังสือทวงถาม)

### AI + ทนาย (semi-automated)

- รับ intake จากลูกค้า → วิเคราะห์ → สรุป → สร้าง lead ใน Odoo
- จัดการนัดหมายปรึกษา
- แจ้งเตือน deadline / นัดศาล

### ทนายต้องทำเอง

- ให้คำปรึกษาคดีซับซ้อน
- ลงนามเอกสารทางกฎหมาย
- ขึ้นศาล / ยื่นฟ้อง

## Knowledge Base (RAG)

เอกสารที่ต้อง upload เข้าระบบ:

| ประเภท | ตัวอย่าง |
|--------|---------|
| ฎีกา | คำพิพากษาศาลฎีกา |
| กฎหมาย | ป.พ.พ., ป.อาญา, ป.วิ.พ. |
| ระเบียบ | กฎกระทรวง, ระเบียบราชการ |
| สัญญาตัวอย่าง | สัญญาซื้อขาย, เช่า, กู้ยืม, จ้างงาน |
| ความรู้ภายใน | อัตราค่าบริการ, ขั้นตอนสำนักงาน |

## Tech Stack

| Layer | เทคโนโลยี |
|-------|-----------|
| AI Agent | adkcode (Google ADK + Gemini) |
| RAG Engine | adkcode rag.py (ดัดแปลงรองรับ PDF) |
| LINE Bot | Odoo controller + LINE Messaging API |
| LIFF | Odoo website/portal pages (QWeb + OWL) |
| CRM / Billing | Odoo 18 |
| Deploy | Docker Compose (Odoo + adkcode) |
